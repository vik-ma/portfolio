# Portfolio Website
This portfolio website was made using **React**, **TypeScript**, **Next.js** and **Tailwind CSS**. The **Parallel Routing** feature introduced in **Next.js 13.3** is used to render the different pages on the site in modals, instead of sending the user to a brand new page!
